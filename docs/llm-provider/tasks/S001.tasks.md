```python
# backend/rag/llm/base.py
from abc import ABC, abstractmethod
from dataclasses import dataclass

@dataclass
class LLMResponse:
    answer: str
    sources: list[str]
    confidence: float
    provider: str
    model: str
    low_confidence: bool

class LLMProvider(ABC):
    @abstractmethod
    async def complete(self, prompt: str, context_chunks: list[str]) -> LLMResponse: ...
```

**review_criteria:**
- `LLMResponse` is a `@dataclass` (not a Pydantic model — internal use only)
- `complete()` is `async` (D07 — full stack is async)
- No provider-specific imports in this file
- `low_confidence` is a field on dataclass, not raised as exception (D04)
