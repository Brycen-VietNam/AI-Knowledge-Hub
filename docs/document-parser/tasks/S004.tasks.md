### T001: Create upload.py route + test_upload.py (unit tests)

**Status**: DONE

**File(s)**

- `tests/api/routes/test_upload.py` — create (unit test cases)

**Change description**
TDD: write unit tests first, then implement.
`router = APIRouter()`.
`PARSER_TIMEOUT_SECS = float(os.getenv("PARSER_TIMEOUT_SECS", "30"))`.
`POST /v1/documents/upload` with `dependencies=[Depends(verify_token)]` (R003, R004).
Form fields: `file: UploadFile`, `title: str | None = Form(None)`, `user_group_id: int | None = Form(None)`, `lang: str | None = Form(None)`.
Handler in order (per plan):
  1. `file` missing → 422 `ERR_NO_FILE`
  2. `SecurityGate.validate(file.size, b"", declared_mime, filename)` (size-first pre-read check)
  3. `content_bytes = await file.read()` then re-validate magic bytes
  4. `ParserFactory.get_parser(...)` → 415 on `UnsupportedFormatError`
  5. `asyncio.wait_for(asyncio.to_thread(parser.parse, content_bytes), timeout=PARSER_TIMEOUT_SECS)`
      → 504 `ERR_PARSE_TIMEOUT` | 422 `ERR_PARSE_FAILED`
  6. `resolved_title = title or Path(file.filename).stem`
  7. `resolved_lang = lang or parsed.lang`
  8. INSERT `Document` row → get `doc_id`
  9. `await audit_log.write(user_id, [doc_id], query_hash=None)` (R006)
  10. `background_tasks.add_task(ingest_pipeline, doc_id, parsed.text)`
  11. `return JSONResponse({"document_id": str(doc_id), "status": "processing"}, 202)`

Unit test cases (mocked parsers/DB):
1. Missing file field → 422 `ERR_NO_FILE`
2. `UnsupportedFormatError` from factory → 415 `ERR_UNSUPPORTED_FORMAT`
3. `SecurityError(code="ERR_FILE_TOO_LARGE")` → 413
4. `SecurityError(code="ERR_MIME_MISMATCH")` → 415
5. `ParseError` from parser → 422 `ERR_PARSE_FAILED`
6. `asyncio.TimeoutError` → 504 `ERR_PARSE_TIMEOUT`

**Review criteria**
- [ ] `dependencies=[Depends(verify_token)]` on route (R003)
- [ ] Route path starts with `/v1/` (R004)
- [ ] `audit_log.write` called before `background_tasks.add_task` (R006)
- [ ] Response `202` with `{"document_id": ..., "status": "processing"}`
- [ ] `PARSER_TIMEOUT_SECS` from env (S005)
- [ ] Error shape: `{"error": {"code": ..., "message": ..., "request_id": ...}}` (A005)
- [ ] `resolved_title` falls back to `Path(filename).stem` (D10)
