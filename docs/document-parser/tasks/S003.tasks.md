### T003: Create SecurityGate + test_security_gate.py

**Status**: DONE

**File(s)**
- `backend/rag/parser/security_gate.py` — create
- `tests/rag/parser/test_security_gate.py` — create

**Change description**
`SecurityGate` (utility class, no base class).
`MAX_UPLOAD_BYTES = int(os.getenv("MAX_UPLOAD_BYTES", str(20 * 1024 * 1024)))`.
`validate(file_size: int, file_bytes: bytes, declared_mime: str, filename: str) -> None`:
  Step 1: `file_size > MAX_UPLOAD_BYTES` → `SecurityError(code="ERR_FILE_TOO_LARGE")` (before reading bytes — D12).
  Step 2: `actual_mime = magic.from_buffer(file_bytes[:2048], mime=True)`.
  Step 3: mismatch check → `SecurityError(code="ERR_MIME_MISMATCH")`.
`_compatible()`: allows `text/plain ↔ text/html` flexibility.
Rejection events logged at `WARNING` via Python `logging` (not `audit_log` table).
TDD: write test first.

**Test cases in test_security_gate.py**:
1. Valid PDF bytes + declared `application/pdf` → no exception
2. `file_size > MAX_UPLOAD_BYTES` with small `file_bytes` → `ERR_FILE_TOO_LARGE` (size check fires before reading bytes — pass small bytes to confirm size arg used)
3. PDF bytes declared as `text/plain` → `ERR_MIME_MISMATCH`
4. `text/plain` bytes declared as `text/html` → no exception (`_compatible`)
5. `MAX_UPLOAD_BYTES` override via env var → respected

**Review criteria**
- [ ] Size checked BEFORE `magic.from_buffer` call (D12 — OOM prevention)
- [ ] `_compatible()` allows `text/plain ↔ text/html`
- [ ] `SecurityError` (not `ValueError`) raised for both error types
- [ ] Rejection logged at `WARNING` not `ERROR`
- [ ] `MAX_UPLOAD_BYTES` reads from env (S005)

**Test command**
```bash
pytest tests/rag/parser/test_security_gate.py -v
```

**Parallel**: safe
**Size estimate**: ~40 lines impl + ~50 lines test
