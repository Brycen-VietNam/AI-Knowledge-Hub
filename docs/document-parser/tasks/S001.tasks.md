### T003: Create ParserFactory (factory.py)

**Status**: DONE

**File(s)**
- `backend/rag/parser/factory.py` — create

**Change description**
`ParserFactory.get_parser(mime_type: str, filename: str) -> ParserBase`.
Primary dispatch by `mime_type`; fallback to file extension only when `mime_type == "application/octet-stream"`.
Supported map: `application/pdf → PdfParser`, DOCX MIME → `DocxParser`, `text/html → HtmlParser`, `text/plain → TxtParser`.
