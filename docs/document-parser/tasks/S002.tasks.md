### T002: Create DocxParser + test_docx_parser.py

**Status**: DONE

**File(s)**
- `backend/rag/parser/docx_parser.py` — create
- `tests/rag/parser/test_docx_parser.py` — create

**Change description**
`DocxParser(ParserBase)`: open bytes via `BytesIO`, extract paragraphs (non-empty text) +
tables (cells joined with `" | "`, rows joined with `"\n"`).
`section_count` = count of paragraphs with `len(text.strip()) > 0`.
`metadata`: `{"section_count": int, "source_format": "docx"}`.
`lang = None` always (A003).
Empty DOCX (no paragraphs/tables) → `ParseError(code="ERR_PARSE_FAILED")`.
Any `python-docx` exception → `ParseError(code="ERR_PARSE_FAILED")`.
TDD: write test first.

**Test cases in test_docx_parser.py**:
1. Multi-section DOCX fixture (paragraphs + 1 table) → text includes both
2. Table cells joined with `" | "` and rows with `"\n"`
3. Empty DOCX (no paragraphs, no tables) → `ERR_PARSE_FAILED`
4. Corrupt/truncated bytes → `ERR_PARSE_FAILED`
5. Performance: ≤200KB fixture < 1s (`@pytest.mark.slow`)

**Review criteria**
