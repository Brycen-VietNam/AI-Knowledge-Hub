## Review Criteria
- `_parse_citations("See [1] and [2].", 3)` → `{0, 1}`
- `_parse_citations("See [99].", 3)` → `set()` (OOB ignored — D-CQ-03)
- `_parse_citations("", 3)` → `set()` (empty answer)
- `_parse_citations("See [1].", 0)` → `set()` (num_docs=0 guard)
- `_parse_citations("[ 1 ]", 3)` → `{0}` (whitespace inside brackets OK)
- stdlib only — no external imports
- sync function — not async
- module importable as `from backend.rag.citation_parser import _parse_citations`
  and via `from backend.rag import _parse_citations`

+"""citation_parser.py — Pure citation marker extraction."""
+import re
+
+def _parse_citations(answer: str, num_docs: int) -> set[int]:
+    if not answer or num_docs <= 0:
+        return set()
+    raw = re.findall(r"\[\s*(\d+)\s*\]", answer)
+    cited: set[int] = set()
+    for token in raw:
+        zero_based = int(token) - 1
+        if 0 <= zero_based < num_docs:
+            cited.add(zero_based)
+    return cited
