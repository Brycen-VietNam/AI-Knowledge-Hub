### T003: Create ResetPasswordModal.tsx component

**Status**: TODO

**File(s)**
- `frontend/src/components/admin/ResetPasswordModal.tsx` — create

**Change description**
New admin modal component. Props: `{ userId: string; onClose: () => void }`.
Mode toggle (radio): "Set manually" | "Auto-generate".

**Auto-generate mode:**
- On submit: `POST /v1/admin/users/{userId}/password-reset` with `{"generate": true}`
- Show returned plaintext password in a copyable read-only field (`.reset-password-generated`)
- Copy button reuses ApiKeyPanel copy-to-clipboard pattern
- Show one-time warning text (`.reset-password-warning`)

**Manual mode:**
- Input: New Password (`<input type="password">`)
- Client validate: `len ≥ 8`
- On submit: `POST /v1/admin/users/{userId}/password-reset` with `{"new_password": "..."}`
- Return 204 → success toast + close modal

Both modes: loading state on submit, error display on failure.

**Review criteria**
- [ ] All `.reset-password-*` CSS classes used (T001)
- [ ] i18n keys via `useTranslation()` (T002)
- [ ] No inline styles (D6)
- [ ] Copy button functional for generated password
- [ ] Loading state on submit
- [ ] Test: render, generate mode, manual mode success, short password error

**Test command**
```bash

```
