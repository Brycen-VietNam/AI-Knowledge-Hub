## SQLAlchemy 2.0 Async Pattern
```python
from sqlalchemy.ext.asyncio import create_async_engine, async_sessionmaker, AsyncSession

engine = create_async_engine(
    os.getenv("DATABASE_URL"),
    pool_size=5,
    max_overflow=15,
    pool_pre_ping=True,
)
async_session_factory = async_sessionmaker(engine, expire_on_commit=False)
```
- `expire_on_commit=False` вЂ” required for async: prevents lazy load errors after commit
- Engine at module level вЂ” NOT inside function (P005)
- URL scheme must be `postgresql+asyncpg://` (D03)
