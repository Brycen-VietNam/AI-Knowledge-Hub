## Knowledge & Lessons Learned

### What Went Well
- Frozen dataclass pattern (`dataclasses.replace()`) được phát hiện sớm trong task analysis — tránh runtime bug
- Security review clean: 0 SQL injection sau 13 queries trong admin.py
- Junction table approach (D08) đúng đắn — normalized, ON DELETE CASCADE clean

### What Could Improve
- Plan gốc có bug (`user.is_admin = ...`) — cần review plan snippet trước khi viết tasks
- f-string SQL pattern nên bị chặn bởi linter rule để future maintainer không copy pattern

### Rule Updates
- Không có rule mới cần thêm — B1 confirms `frozen=True` dataclass phải dùng `dataclasses.replace()`
