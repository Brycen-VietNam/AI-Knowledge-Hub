### T004: verify_token — compute is_admin via helper

**Status:** TODO | **Agent:** api-agent

**File(s)**
- `backend/auth/dependencies.py` — modify

**Change description**
Thêm async helper `_compute_is_admin(user_id: uuid.UUID, db: AsyncSession) -> bool` dùng parameterized SQL JOIN trên `user_group_memberships → user_groups`. Trong `verify_token`, sau khi bất kỳ auth path nào trả về `user`, gọi helper và rebuild với `dataclasses.replace(user, is_admin=is_admin)`.

> ⚠️ **Bug trong plan gốc:** Plan snippet dùng `user.is_admin = result.scalar()` — sẽ raise `FrozenInstanceError` vì `AuthenticatedUser` là `frozen=True`. Phải dùng `dataclasses.replace()`.

SQL cho helper:
```sql
SELECT BOOL_OR(ug.is_admin) AS is_admin
FROM user_group_memberships ugm
JOIN user_groups ug ON ug.id = ugm.group_id
WHERE ugm.user_id = :user_id
```

**Review criteria**
- [ ] Dùng `dataclasses.replace(user, is_admin=...)` — không mutate frozen dataclass
- [ ] `import dataclasses` thêm vào top of file
- [ ] SQL dùng `text(...).bindparams(...)` — no f-string (S001)
- [ ] `BOOL_OR` aggregate: `result.scalar() or False` — handle NULL khi user không thuộc group nào
- [ ] Helper gọi cho TẤT CẢ auth paths (api_key, local JWT, OIDC) — một điểm duy nhất
- [ ] Rule S001: parameterized SQL
