### T001: ApiKeyCreate model + key generation logic
**Status**: DONE ✅

**File(s)**
- `backend/api/routes/admin.py` — modify: add `ApiKeyCreate` model + `_generate_api_key()` helper

**Change description**
Add `ApiKeyCreate(BaseModel)` with one optional field: `name: Optional[str] = Field(None, max_length=100)`.
Add private helper `_generate_api_key() -> tuple[str, str, str]` returning
`(plaintext, key_hash, key_prefix)`:
- `plaintext = "kh_" + secrets.token_hex(16)` (36 chars)
- `key_hash = hashlib.sha256(plaintext.encode()).hexdigest()`
- `key_prefix = plaintext[:8]`
Add imports: `import secrets`, `import hashlib`.

**Review criteria**
- [ ] `name` max_length=100 enforced by Pydantic
- [ ] `plaintext` format: `kh_` + 32 hex chars = 36 chars total
- [ ] `key_prefix` is exactly `plaintext[:8]` (first 8 chars of key, not hash)
- [ ] `hashlib.sha256` used — not md5 or sha1 (S005)
- [ ] Helper is private (`_generate_api_key`) — not exported

### T002: POST handler — user check, INSERT api_key, one-time plaintext return
**Status**: DONE ✅

**File(s)**
- `backend/api/routes/admin.py` — modify: add `admin_generate_api_key` route

**Change description**
Add `@router.post("/v1/admin/users/{user_id}/api-keys", dependencies=[Depends(require_admin)], status_code=201)`.
Handler: verify user exists → 404; call `_generate_api_key()`; INSERT into `api_keys`
`(id, user_id, key_hash, key_prefix, name, created_at)` via `text().bindparams()`.
Return 201: `{key_id, key: plaintext, key_prefix, name, created_at}`. Plaintext never
stored in DB — only returned here once.

**Review criteria**
- [ ] Route under `/v1/admin/` with `require_admin` (R003, R004)
- [ ] 404 if user not found before any INSERT (A005 shape)
- [ ] `key_hash` stored — NOT `key` plaintext (S005)
- [ ] Response includes `key` (plaintext) — caller must save it now
- [ ] `key_hash` and `key_prefix` stored via `text().bindparams()` (S001)
- [ ] `name` stripped of leading/trailing whitespace before store (S003)
