### T002: DELETE handler — revoke api-key
**Status**: TODO

**File(s)**
- `backend/api/routes/admin.py` — modify: add `admin_revoke_api_key` route

**Change description**
Add `@router.delete("/v1/admin/users/{user_id}/api-keys/{key_id}", dependencies=[Depends(require_admin)])`.
Handler: verify user exists → 404; verify key exists for that user → 404;
DELETE from `api_keys WHERE id = :key_id AND user_id = :uid` via `text().bindparams()`.
Return `200 {"revoked": str(key_id)}`.

**Review criteria**
- [ ] Route under `/v1/admin/` with `require_admin` (R003, R004)
- [ ] 404 for unknown user (A005)
- [ ] 404 for unknown key_id (A005) — must belong to that user (prevents key enumeration)
- [ ] WHERE clause includes BOTH `key_id AND user_id` — not just key_id (S001 + isolation)
- [ ] SQL parameterized (S001)
- [ ] Success: `200 {"revoked": "<key_id>"}` exactly
