#### ‚öÝÔ∏è WARNING ‚Äî W002: embed() call outside timeout wrapper
**File**: `backend/api/routes/query.py` lines 105-114
`await embed(body.query)` executes before `asyncio.wait_for(retrieve(...), timeout=1.8)`. If the embedding call is slow, the total time can exceed the 1.8s SLA without triggering the timeout. Currently `embed()` is a zero-cost stub, so no immediate risk. When the real embedder is wired, both calls should be inside the timeout or `embed()` should have its own timeout.
**Severity**: Low ‚Äî stub only; tracked for embedder integration story.

#### ‚öÝÔ∏è WARNING ‚Äî W003: Return type annotation mismatch
**File**: `backend/api/routes/query.py` line 93
`-> QueryResponse` declared but the timeout path returns `JSONResponse`. This is valid FastAPI behavior (Response subclass bypasses response_model), but the type annotation is technically inaccurate. Consider `-> QueryResponse | JSONResponse` or `-> Response`.
**Severity**: Low ‚Äî runtime correct; type checker may flag.
