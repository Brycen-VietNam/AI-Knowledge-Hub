## Patterns to Follow

### SQL style (SECURITY.md S001 — HARD BLOCK)
```python
# CORRECT — text().bindparams() throughout
from sqlalchemy import text, bindparam
sql = text("SELECT ... WHERE id = :id").bindparams(bindparam("id", value=some_id))
# WRONG — never use f-strings in SQL
sql = f"SELECT ... WHERE id = {some_id}"  # ❌
```

## RBAC WHERE Clause — Critical Design Points

### Dense path (T002)
Filter on `embeddings.user_group_id` — embeddings table is the query starting point.
```sql
WHERE (e.user_group_id = ANY(:group_ids) OR e.user_group_id IS NULL)
```
- `ANY([])` in PostgreSQL = `FALSE` → OR IS NULL branch still matches public docs ✅
- Partial index `idx_embeddings_public` (migration 005) accelerates the IS NULL branch

### BM25 path (T003)
Filter on `documents.user_group_id` — documents table is the BM25 query starting point (D02).
```sql
WHERE (d.user_group_id = ANY(:group_ids) OR d.user_group_id IS NULL)
  AND d.content_fts @@ to_tsquery('simple', :query)
```
- RBAC WHERE must come BEFORE ORDER BY / LIMIT (R001/AC4)
- Partial index `idx_documents_public` (migration 005) accelerates the IS NULL branch

### Hybrid merge (T004)
- `asyncio.gather` for concurrent dense + BM25 (P001 latency SLA)
- `asyncio.wait_for(..., timeout=1.8)` wraps full call (R007)
- Weights from env: `RAG_DENSE_WEIGHT=0.7`, `RAG_BM25_WEIGHT=0.3` (A004)
- Dedup by `doc_id`, weighted score sum
