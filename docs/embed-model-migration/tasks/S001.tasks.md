- [ ] Rule satisfied: P002 (single call still uses async client correctly)
- [ ] **X3 — exact byte-level prefix**: test asserts outgoing prompt starts with `"query: "` (7 chars: `q-u-e-r-y-:-SPACE`), NOT variants like `"Query:"`, `"query:"` (no space), or `"q: "`. Use `assert prompt.startswith("query: ")` and `assert prompt[7] != " "` (no double-space).
