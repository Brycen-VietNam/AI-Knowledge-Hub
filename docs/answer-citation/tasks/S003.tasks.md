**Change description**
Replace existing prompt template with numbered-index format verbatim from spec AC3.

New template structure:
```
You are a helpful assistant. Answer the user's question using only the provided sources.
When referencing a source, cite it inline using [N] where N is the source number.

Sources:
{sources_index}

Question: {question}

Answer:
```

Where `{sources_index}` is rendered at call site as:
```
[1] {title_1}
```

`{sources_index}` + `{question}` are volatile suffix (Route A compliant).
